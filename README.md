A simple web-based basketball statistics tracker built with Flask. This app allows users to view and manage player stats such as points, rebounds, and assists.

Features:
    Track points, rebounds, and assists
    Calculate averages (PPG, RPG, APG) (if implemented)
    Simple and responsive web interface
    Easy to expand with new features

Live Demo deployed with Render:
    https://statswebapp.onrender.com/

Built With:
    Python 3
    Flask
    HTML (Jinja2 templates)
    Gunicorn (for deployment)
